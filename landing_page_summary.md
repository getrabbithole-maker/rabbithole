# Landing Page Summary: Rabbithole

The Rabbithole landing page is designed with a premium, focused aesthetic (glassmorphism/dark mode by default) centered around the concept of "Deep Work" and "Flow State".

## 1. Visual Structure & Components
- **Header**: Mini navigation with a minimalist Theme Toggle (Sun/Moon) and Language Switcher (EN/ไทย).
- **Hero Section**: 
  - Centered Logo (Rabbithole).
  - Main Title: "Rabbithole".
  - Subtitle: "Go deep. Stay deep." (Dynamic font based on language: handwriting for TH, cursive for EN).
  - Description: Emphasizes a 90-minute focus ceremony with immersive audio.
- **Primary CTAs**: 
  - "Enter the Rabbit Hole" (Link to Dashboard).
  - "Pricing" (Scrolls to pricing section).
- **Showcase Carousel**: An interactive section showing the app's interface (Dashboard, Planning, Focus Station) with descriptive tags (e.g., "Analog", "Immersive Audio").
- **How It Works**: A three-step guide: 1. Plan, 2. Isolate, 3. Review.
- **Benefits**: Bullet points emphasizing time reclamation, science-backed habits, and ambient soundscapes.
- **Reviews**: Grid of user feedback with star ratings and author roles.
- **Pricing Table**: Comparison between "Deep Free" and "Ritual Pro" (1.99 USD / 39 THB).
- **Footer**: Copyright, Legal links (Privacy/Terms), and acknowledgment of the creator ("Built by Namwaaok").

## 2. Technical Features
- **i18n Support**: Full support for English and Thai.
- **Theme Support**: Dark/Light mode toggle.
- **Animations**: Uses intersection observers for fade-in/up effects as the user scrolls.
- **Responsive Design**: Mobile-optimized layout with adjusted font sizes and carousel behavior.
